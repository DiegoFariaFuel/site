PERSONALIZAÇÕES POSSÍVEIS
═══════════════════════════════════════════════════════════════════════════

Se quiser adicionar mais idiomas no futuro:

1. Abra o arquivo "solviz (2).html"
2. Procure por "const languages = ["
3. Adicione uma nova linha seguindo o padrão:
   { code: 'CÓDIGO', name: 'NOME', flag: '🏳️', dir: 'ltr' }

Exemplo para adicionar Catalão:
   { code: 'ca', name: 'Català', flag: '🏴', dir: 'ltr' }

SOLUÇÃO DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════

❌ "A tradução não acontece":
   → Verifique sua conexão com internet
   → Limpe o cache do navegador (Ctrl+Shift+Del)
   → Tente outro navegador

❌ "O menu de idiomas não abre":
   → Verifique se JavaScript está habilitado
   → Atualize a página (F5)

❌ "Tradução parcial ou incorreta":
   → Isso é normal com tradução automática
   → Google Translate melhora com o tempo
   → Para traduções profissionais, contrate um tradutor
